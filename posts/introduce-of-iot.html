Introduce of iot

发表于 2020-03-01 更新于 2020-03-02 分类于 Iot 本文字数： 930 阅读时长 ≈ 1 分钟
  • 物联网：

通过射频识别（RFID）装置、传感器、 全球定位系统等信息传感设备，按照约定的协议，把地球上的任何具体事物同互联网连接起来，并进行信息交换和通信，实现识别、定位、监控、控制和管理的一种网络。

定义

  • 物：
    • 有相应的接收器
    • 发送器
    • cpu
    • 存储功能
    • 操作系统
    • 数据发送器
    • 专门的应用程序
  • 联
    • 物物相连的互联网
    • 众多传感设备连接起来并在广域网内今昔信息
  • 网
    • 基础和核心依旧是互联网，却远远超出互联网的范畴。
  • IOT的三个特性
    • 不能简单看为互联网的延伸
    • 伴随新的业务共同发展
    • 包括不同的通信模式，强调M2M（man to machine | machine to machi…. ）

2. 特征

  • 全面感知 （核心）
    • 利用传感器、电子标签等环境的各个可测量特征
  • 可靠传递
    • 数据传递的稳定性和可靠性
    • adhoc多跳上传数据
    • 有线和无线
    • zigbee，lora，nbIot
  • 智能处理（研究热点）
    • 目的：智能识别跟踪，定位监控和管理等
    • 云计算和人工智能对海量数据进行存储分析和处理

3. 体系结构（重要）

桂晓琳（西交）五层架构：

  • 数据存储
  • 数据处理
  • 通信处理
  • 自动控制技术
  • 标识和感知

三层架构（或四层）

  • 海 传感器海量的感知层。
    在感知层进行计算，也是一个研究方向

    • 物计算：在感知层对数据进行计算（轻量级）

    • 雾计算：边缘计算，分担云的计算压力

  • 网 互联网

    互联网形态很多，如何整合物联网到各个网络上进行应用？

    ngi（next generation Internet） 下一代网络
    Wikipedia : NGI

    无线网络 LTE

    与互联网隔离的专用网络，保证安全

    需要根据每一个层次的每一个模块进行安全处理

  • 云 应用处理层

分布式云

服务层也常称为中间件

4. 关键技术

  • 基础资源管理与服务 pki

    • 名称和地址（依托IPV6）
    • 核心技术

  • 信息获取和识别技术

    • 传感器
    • RFID

  • 组网与传输技术

    • 无线传感网
    • 计算机网络

  • 信息处理技术

    传统数据库无法满足信息存储需求

    • 数据挖掘
    • 数据清洗
    • 深度学习
    • 云和虚拟化
    • …

  • 安全技术

  • 物联网系统集成技术

    • MEMS（微电机系统）
    • 集感知传输预处理等功能集中在一体的节点设备

  • 共性支持技术

    • 专用操作系统和综合软件开发环境

5. 应用和发展趋势

  • 安全防护
  • 更普及的智能消费设备
  • 更加关注人工智能
  • 更快速的数据转化

6. 物联网行业

  • 智慧家居
  • 智慧农业
  • 智慧医疗
  • 智慧工业
  • 智慧教育
  • 食品安全
  • 智慧城市
  • 智慧交通
  • …

物联网是一种趋势，更多的需要我们去探索和创造…
